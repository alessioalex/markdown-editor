# markdown-editor ![pic](screenshot.png) Markdown editor with live preview, inspired by [thlorenz/browserify-markdown-editor](https://github.com/thlorenz/browserify-markdown-editor) ## opening the editor ```bash # build the browserify bundle and start the server # on `process.env.PORT` or 7777 by default npm run engage ``` Now open [localhost:7777](http://localhost:7777) and have some markdown fun. ## License MIT
